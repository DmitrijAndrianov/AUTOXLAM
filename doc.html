Главная страница

Давайте посмотрим на структуру Целевой страницы. Она включает в себя собственные разделы с контентом. По запросу вы можете добавлять или удалять разделы сайта. Пожалуйста, будьте осторожны при удалении ненужных разделов. Это может вызвать ошибки в theme.js. Все теги должны быть закрыты правильно.

Инструменты в помощь:

  • Firebug
  • Chrome devtools

У каждого раздела есть свой класс, который влияет на отображение контента.

Для некоторых разделов мы используем дополнительные скрипты, которые находятся в папке PLUGINS.

Все скрипты подключаются в подвале сайта.

Настройки переключателя цветов

Установите атрибут data-default-color="true", чтобы изменить цвет по умолчанию

assets/switcher/ 

<link rel="alternate stylesheet" type="text/css" href="assets/switcher/css/color1.css" title="color1" media="all"
data-default-color="true" />
